# XORKey

---
About:
A Python tool designed to encrypt arbitrary strings using XOR operations, offering multiple encryption configurations such as customizable keys, encoding formats, and output variations.

## Features

- Core XOR encryption/decryption functions (`xorkey.core`)  
- Utility functions for key management and data manipulation (`xorkey.utils`)  
- Command-line interface for quick operations (`xorkey.main`)  
- Modular design for easy integration
