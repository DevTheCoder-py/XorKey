# XORKey

A Python library for working with XOR-based encryption and related utilities. Designed for simplicity, modularity, and ease of integration into other Python projects.

## Features

- Core XOR encryption/decryption functions (`xorkey.core`)  
- Utility functions for key management and data manipulation (`xorkey.utils`)  
- Command-line interface for quick operations (`xorkey.main`)  
- Modular design for easy integration

## Installation

### Linux
```
cd ~
git clone https://github.com/DevTheCoder-py/XorKey
python -m venv env
source env/bin/activate # if you are using fish/csh use source env/bin/activate.fish OR env/bin/activate.csh
pipx install -e .
# All Done!
```
